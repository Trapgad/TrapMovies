# TRAP MOVIES

A modern, responsive movie and series discovery platform built with HTML, CSS, and JavaScript. This project fetches real-time data from The Movie Database (TMDb) API to provide users with the latest trending content.

## Features
* **Dynamic Content**: Fetches real-time movies and series data.
* **Multi-Page Layout**: Includes dedicated pages for Movies, Series, About, and Contact.
* **Responsive Design**: Designed to work on mobile, tablet, and desktop devices.
* **Search Functionality**: Allows users to find their favorite titles quickly.

## Technologies Used
* **Frontend**: HTML5, CSS3, Vanilla JavaScript.
* **API**: The Movie Database (TMDb) API.
* **Deployment**: Hosted on Vercel.

## How to Run
1. Clone this repository.
2. Get your API Key from [TMDb](https://www.themoviedb.org/).
3. Add your API Key to the `script.js` file.
4. Open `index.html` in your browser.
